EBookify — Digital Reading & Publishing Platform

EBookify is a modern full-stack web application designed for reading, exploring, and publishing digital books. It combines a beautiful futuristic UI with smooth interactions, secure authentication, a dynamic library, and an immersive in-browser reading experience.
Features:
        User Authentication:
                1.Secure signup & login using JWT + bcrypt
                2.Cookie-based authentication
                3.Form validation & error handling
        Reader Mode:
                1.Fully immersive HTML-based reading interface
                2.Adjustable font size
                3. Light / Dark 
                4. Smooth page slider
                5. Responsive, distraction-free design
        Explore Books:
                1.Trending section
                2.Genre filters (Fiction, Science, etc.)
                3.Search functionality
                4.Detailed book cards with rating, language & cover
        Book Details Page:
                1.Synopsis, ratings, reviews
                2."Read Now" → Opens the interactive reader
                3."Add to Library" button
                4.Responsive layout & animations
        Publish Books:
                1.Upload own books as PDF/EPUB
                2.Fill metadata: title, author, genre, synopsis
                3.Cover image upload
                4.Beautiful form UI with glowing upload box
